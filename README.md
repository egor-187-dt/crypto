# CryptoSafe Manager 🔐

Менеджер паролей с безопасным шифрованием. Храни свои пароли в зашифрованном виде с мастер-паролем.

## ✅ Sprint 1 (Базовая функциональность)
- ✅ Мастер-пароль и вход в систему
- ✅ Хранение записей (логин/пароль) в SQLite
- ✅ Система событий (event bus)
- ✅ Очистка памяти от ключей
- ✅ Несколько баз данных
- ✅ Простое шифрование (XOR, для тестов)
- ✅ Базовые тесты

## ✅ Sprint 2 (Безопасность)
- ✅ **Argon2id** для хеширования мастер-пароля (рекомендуемый стандарт)
- ✅ **PBKDF2-HMAC-SHA256** для вывода ключа шифрования
- ✅ Соли для каждого пользователя (16 байт)
- ✅ Экспоненциальная задержка при неудачных попытках входа
- ✅ Блокировка на 30 секунд после 5 неудачных попыток
- ✅ Проверка сложности пароля (минимум 12 символов)
- ✅ Безопасная очистка ключей из памяти
- ✅ Блокировка страниц памяти (VirtualLock/mlock)
- ✅ Автоблокировка при неактивности
- ✅ Смена мастер-пароля с перешифровкой всех записей
- ✅ Миграция БД без потери данных
- ✅ Совместимость со старыми БД (SHA256)

# Sprint 3

Менеджер паролей с AES-256-GCM шифрованием и безопасным хранением данных.

## Возможности

-  **AES-256-GCM шифрование** каждой записи с уникальным nonce
-  **Argon2id** хеширование мастер-пароля
-  **CRUD операции** с записями (создание, чтение, обновление, удаление)
-  **Поиск** в реальном времени по названию и логину
-  **Генератор паролей** с проверкой сложности
-  **Автоблокировка** при бездействии (60 минут)
-  **Резервное копирование** базы данных
-  **Поддержка нескольких БД**
